# Parse on .NET 3.5

* Parse Plugins for .NET 3.5
* Depends on Firebase

## Requirement

* Unity
* **.NET 3.5**

## Install

```shell
yarn add "umm-projects/parse_dotnet35#^1.0.0"
```

## Usage

* Use this module if your project depends Firebase and setting of `Scripting Runtime Version` is **.NET 3.5**.
